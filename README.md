## Descripción del proyecto

Este repositorio contiene el desarrollo progresivo de las actividades prácticas realizadas durante la asignatura de Desarrollo de Aplicaciones Web. A lo largo del curso, el proyecto evolucionó desde una página web básica construida con HTML hasta un sitio más completo, incorporando nuevas secciones, recursos multimedia y una estructura organizada mediante HTML5.

La versión actual presenta una página web desarrollada con fines académicos, integrando contenido informativo, elementos visuales y recursos multimedia que permiten mejorar la experiencia de navegación y la presentación de la información.

## Estructura del proyecto

El proyecto está compuesto por:

- Archivo principal `index.html`.
- Carpeta de imágenes y recursos multimedia.
- Estructura basada en HTML5.
- Encabezados jerárquicos (`h1`, `h2` y `h3`).
- Menú de navegación.
- Secciones de contenido organizadas de forma semántica.
- Información de contacto.
- Estilos básicos implementados con CSS.

## Avances del proyecto

### Semana 2

- Creación de la estructura inicial del documento HTML.
- Uso de encabezados para organizar la información.
- Definición del tema principal del proyecto.
- Aplicación de conceptos básicos de diseño web.

### Semana 3

- Implementación de un menú de navegación interno.
- Desarrollo de diferentes secciones de contenido.
- Inserción de imágenes relacionadas con la temática del sitio.
- Incorporación de contenido multimedia.
- Uso de etiquetas semánticas como `header`, `nav`, `main`, `section`, `aside` y `footer`.
- Mejora de la organización y presentación visual mediante CSS.

### Semana 4 
- **Integración de Bootstrap:** Vinculación del framework mediante CDN en el `head`.
- **Diseño Responsivo:** Implementación del sistema de grillas (`container`, `row`, `col`) para asegurar la adaptabilidad en pantallas móviles, tablets y computadoras.
- **Componentes Avanzados:** 
  - Transformación del menú en una barra de navegación responsiva (`navbar`).
  - Presentación de servicios o productos mediante tarjetas estilizadas (`card`).
  - Optimización de imágenes utilizando la clase `img-fluid`.
- **Sección de Contacto:** Inclusión de un formulario estructurado con campos de texto (`form-control`) para nombre, correo, asunto y mensaje, además de botones para redes sociales y ubicación.
- **Estilos CSS3 Personalizados:** Definición de tipografías, paleta de colores, márgenes, sombras (`box-shadow`), efectos visuales y aplicación de *Media Queries* para ajustes finos en pantallas pequeñas.

## Tecnologías utilizadas

- HTML5 (Estructura semántica)
- CSS3 (Estilos personalizados y Media Queries)
- Framework Bootstrap (Sistema de grillas y componentes responsivos)
- Recursos multimedia adaptables (Imágenes y video)
- Control de versiones con GitHub y despliegue en GitHub Pages

## Objetivo del proyecto

Aplicar de manera práctica los conocimientos adquiridos en la asignatura para diseñar y desarrollar una aplicación web funcional, estructurada y estéticamente organizada. El enfoque central de esta etapa es dominar el diseño responsivo (*Mobile First*) y la combinación eficiente de estilos CSS3 propios con los componentes estandarizados de Bootstrap, garantizando una excelente experiencia de usuario (UX) en cualquier dispositivo.
